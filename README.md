# spring-gift-enhancement

### 4주차 step0

---
## 할 일
1. 상품 정보에 카테고리 추가
    - 상품과 카테고리 모델 간의 관계를 고려하여 설계하고 구현
2. 상품에는 항상 하나의 카테고리가 있어야 함
3. 상품 카테고리는 수정할 수 있음
4. 관리자 화면에서 상품을 추가할 때 카테고리를 지정할 수 있음
5. 1차 카테고리만 존재하며 2차 카테고리는 고려하지 않음

## 기능 구현 목록
1. 카테고리 엔티티 추가
2. 상품 엔티티에 카테고리 필드 추가 및 매핑
3. 카테고리 리포지토리 추가
4. 카테고리 서비스 추가
5. 카테고리 컨트롤러 추가
6. 상품 추가 및 수정 시 카테고리 지정 기능 추가
7. 카테고리 관련 테스트 코드 작성
